# System Management Data Pokok Siswa

Sebuah sistem pengelola data pokok siswa yang diperuntukkan ujian akhir semester kelas 12 

## Stack
- Laravel
- React
- Inertiajs

## Install
Install project
```
git clone https://github.com/alf4ridzi/dapos
cd dapos
composer install
npm install
```

Jalankan project
```
composer run dev
```
